# Instalar Guest Additions en Debian

Hoy veremos como instalar los drivers de **VirtualBox** para una maquina virtual donde tengamos instalado la distribución "Debian" sin entorno gráfico. 
Este paquete llamado "**Guest Additions**" nos va a permitir aprovechar las características de VirtualBox y mejorar el rendimiento de la maquina virtual, añadiendo funcionalidades y drivers al sistema. 
Vamos a partir de la base que supone tener una maquina virtual con la distribución "Debian 9.6 64 Bits" instalada sin entorno gráfico.

Primero introduciremos el CD de la "_Guest Additions_" desde la barra superior en la ventana de la maquina virtual. Se hace clic en "_Dispositivos > Insertar imagen de CD de las "Guest Additions_"...". Una vez introducida la imagen del CD, entraremos como el usuario "_root_" para poder tener permisos de administrador e introduciremos los siguientes comandos:

```jsx
apt update && apt install linux-headers-amd64 make gcc perl -y
mount /dev/sr0 /media/cdrom0
mount -o remount,ro,exec /dev/sr0
/media/cdrom/VboxLinuxAdditions.run
umount /dev/sr0
reboot
```

Una vez hayamos introducido los comandos, reiniciamos la maquina y ya tendríamos instaladas y configuradas las "_Guest Additions_".
Podremos comprobar los módulos añadidos a nuestro sistema con el siguiente comando:

```jsx
lsmod | grep "vbox"
```

Este tutorial puede ser encontrado en bastantes páginas web, aunque es cierto que hay distintas formas de realizarlo. Espero que mi explicación os haya servido de ayuda.
